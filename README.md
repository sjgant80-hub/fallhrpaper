## For the end user

Generate, preview, and issue UK HR documents from 8 templates. Locked clauses preserve statutory wording (ERA 1996, ACAS Code, settlement agreement s.203 waiver). Editable sections are the employer's responsibility.

### Templates

| Code | Template | Category |
|---|---|---|
| OFR-01 | Offer Letter | Pre-employment |
| S1-01 | S.1 Statement of Employment Particulars | Core |
| EC-01 | Employment Contract | Core |
| DISC-01 | Disciplinary Hearing Invite | Discipline |
| GR-01 | Grievance Outcome Letter | Discipline |
| RED-01 | Redundancy Notice | Termination |
| SA-01 | Settlement Agreement (s.203) | Termination |
| REF-01 | Employment Reference | Post-employment |

### Key rules

- **ERA 1996 s.1**: S.1 statement on or before day one
- **ACAS Code**: disciplinary/grievance procedure, right to be accompanied, right of appeal
- **ERA 1996 s.203**: settlement agreement waives statutory claims only if independent adviser involved
- **Spring v Guardian Assurance**: references — duty of care to both subject and recipient
- **Income Tax (Earnings and Pensions) Act 2003 s.401**: £30k tax-free on termination payments
